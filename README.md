## 💡 Summary

- 📍 FMCW 레이더를 활용하여 모빌리티 환경에서도 다중 사용자의 생체신호(심박수, 호흡)를 정밀하게 추출
- 📍 잡음 제거 (Static Clutter, DC Drift, Jitter) 및 VMD 기반 신호 분리 기법 제안
- 📍 실제 환경 (실내/열차)에서 측정한 결과와 HR Monitor 데이터 비교

다중 대상 생체신호 감지 기술을 모빌리티에 적용한 성과입니다.
